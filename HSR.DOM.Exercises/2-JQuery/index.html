Performance

Das folgende Code-Stück zeigt ein typisches jQuery Fragment. Finden Sie Optimierungsmöglichkeiten in Bezug auf die Performance.

Tipps:

  • Der jQuery Selektor ist relativ aufwändig.
  • jQuery Funktionen können aneinander gehängt werden (function chaining).
  • Die jQuery Selektor-Funktion unterstützt mehrere Argumente.


$('#frmFeedback .submit').click(function() {
    $('#lblFormHeader').blur();
    $('#lblFormHeader').hide();
    $('#frmFeedback .state').css('background-color', 'red');
});	


var feedbackForm = $('#frmFeedback');
$('.submit', feedbackForm).click(function() {
	$('#lblFormHeader').blur().hide();
	$('.state', feedbackForm).css('background-color', 'red');
});
